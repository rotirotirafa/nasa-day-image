# nasa-day-image
Utilizando a APOD-API da NASA para exibir a imagem do dia.

Fonte: [NASA] - https://api.nasa.gov/index.html

# Instalação:
    npm install

# Gulp
    npm install -g gulp

# Build

    Estes builds são provisórios pois ainda serão melhorados e otimizados.

    gulp js

    gulp angular

    gulp angular-resource

    gulp css 

    gulp bulma
